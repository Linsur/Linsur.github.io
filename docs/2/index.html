Classification

通过前面的逻辑回归模型，可以预测一个介于0到1之间的值表示某件事情发生的概率，例如预测电子邮件为垃圾邮件的概率为75%。

阈值

我们想要部署此模型，将邮件分为垃圾邮件与非垃圾邮件两类。涉及二分类问题时，我们需要选择一个分类阈值作为邮件划分的基准，高于阈值将被分配到正类别，低于阈值分配到负类别。

 

混淆矩阵

二元分类器每个输出都有四种可能情况。

每行表示所有预测正例（TP + FP）和所有预测负例（FN + TN）

每列表示所有真实正例（TP + FN）和所有真实负例（FP + TN）

如果实际正例和实际负例综述不接近，则表示数据集不平衡。

 

一般来说，提高分类阈值将会让真假正例均降低，让真假负例均增加。

召回率

假正例率

假正例率（FPR）：是指被错误归类为正例的所有实际负例所占的比例，也成为误报概率。

数学定义为：

在实际负例非常少的不平衡数据集中，假正例率作为指标意义不大。

精确率和召回率通常呈反函数关系。

预测偏差

导致预测偏差的原因可能是：

  • 数据中的噪声
  • 过度正则化
  • 特征集不足以拟合模型
