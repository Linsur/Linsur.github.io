损失

损失是一个数值指标，损失函数用于衡量模型预测效果精准度。

训练模型的目标是尽可能降低损失，将其降至最低值。

 

丢失距离

损失函数侧重于值之间的距离。

有两种常用方法。

  • 计算预测值与实际值的绝对值。
  • 将预测值与实际值误差平方。
